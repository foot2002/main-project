

# WiseIN Company 기업 웹사이트 개발 계획

## 프로젝트 개요
23년 역사의 데이터 & AI 전문 기업 WiseIN의 신뢰 중심 기업 웹사이트를 구축합니다. 공공기관 및 대기업 의사결정자를 타겟으로, 글로벌 수준의 세련된 기업 이미지를 전달합니다.

> 웹사이트 언어: **한국어** (영어 서브타이틀 병행)

---

## 디자인 방향
- **컬러**: 화이트 베이스 + 딥 네이비 주색 + 블루 하이라이트
- **레이아웃**: 충분한 여백, 구조적 그리드, 대칭적 배치
- **톤**: 절제된 미니멀리즘, 신뢰 중심, 공공 친화적
- **타이포그래피**: 깔끔한 국제적 느낌의 서체
- **애니메이션**: 숫자 카운트업 등 최소한의 인터랙션만 적용

---

## 페이지 구성 (총 6개 섹션)

### 1. HOME
- **히어로 섹션**: "데이터와 AI로 비즈니스의 세계를 열다" 메인 카피, 다크 네이비 배경 이미지 + 오버레이, CTA 버튼 2개 (서비스 영역 / 포트폴리오)
- **신뢰 지표**: 23년 경험 / 2,000+ 프로젝트 / 1,400+ 고객사 / 30,000+ 회원 (카운트업 애니메이션)
- **회사 소개 요약**: 전략적 소개 문구 + About Us 링크
- **핵심 사업 요약**: 데이터 분석, AI 솔루션, 플랫폼 서비스, 교육 — 4개 카드
- **고객사 섹션**: 공공기관, 지자체, 금융, 제조, 교육, 스타트업 업종별 텍스트 분류
- **최종 CTA**: "데이터와 AI가 중요할 때, WiseIN이 답합니다." + 포트폴리오 버튼

### 2. Business (사업 영역)
- **사업 총괄 개요**: 전략적 파트너 포지셔닝
- **데이터 분석**: 공공 데이터 분석 / 기업 데이터 전략 / 리서치 & 컨설팅 (각 개요·접근방식·기대효과)
- **AI 솔루션**: 설문조사 자동화 SaaS / AI 마케팅 자동화 (각 개요·주요기능·기술기반·기대효과)
- **플랫폼 서비스**: Richway 플랫폼 (개요·주요기능·차별점)
- **교육**: 데이터 & AI 교육 프로그램, 정부지원과정 (간략 구성)

### 3. Portfolio (포트폴리오)
- **목록 페이지**: 업종 필터 + 서비스 유형 필터 + 카드 레이아웃
- **상세 페이지 템플릿**: 프로젝트 개요 → 배경 & 과제 → 솔루션 접근 → 수행 내용 → 성과 & 임팩트
- 정량적 성과 중심 표현
- *CMS 연동 없이 정적 샘플 데이터로 구현 (추후 백엔드 연결 가능)*

### 4. Blog (블로그)
- **카테고리**: Data Insight / AI Technology / Industry Trend / Company News
- **목록 페이지**: 카테고리 필터 + 카드 레이아웃
- **상세 페이지 템플릿**: 제목 → 요약 → 본문 → "WiseIN Insight" 전략적 시사점
- *정적 샘플 데이터로 구현*

### 5. About Us (회사 소개)
- 회사 개요
- 미션 & 비전
- 연혁 (타임라인)
- 언론 보도
- 오시는 길

### 6. Support (고객 지원)
- **문의하기**: 입력 폼 (이름, 이메일, 전화번호, 문의 유형, 내용)
- **FAQ**: 아코디언 형태
- **공지사항**: 목록 + 상세 페이지 (정적 샘플)

---

## 공통 요소
- **헤더**: 로고 + 메가 네비게이션 + 모바일 햄버거 메뉴
- **푸터**: 회사 정보, 사이트맵 링크, 저작권
- **반응형**: 데스크톱 / 태블릿 / 모바일 완전 대응
- **접근성**: 시맨틱 마크업, 충분한 색 대비

## 참고 사항
- 포트폴리오, 블로그, 공지사항은 샘플 데이터로 구현합니다. 실제 CMS 기능이 필요한 경우 추후 Supabase 백엔드를 연결할 수 있습니다.
- 문의 폼은 프론트엔드 유효성 검증까지 구현하며, 실제 전송 기능은 백엔드 연결 시 활성화됩니다.
